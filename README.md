# Fraud Detection in Financial Transactions using Neural Networks and Ensemble Learning

## 📌 Dissertation Topic
**How AI Coding Agents Modify Code: A Large-Scale Study of GitHub Pull Requests**

---

## 🎯 Project Overview
This project focuses on detecting fraudulent financial transactions using advanced machine learning techniques, including neural networks and ensemble learning methods. 

In addition, the study explores how AI coding agents modify code by analyzing GitHub pull requests, providing insights into automated code generation and modification patterns.

---

## 🧠 Objectives
- Detect fraudulent transactions with high accuracy
- Apply neural networks for deep pattern recognition
- Use ensemble models to improve prediction performance
- Analyze GitHub pull requests to understand AI-driven code changes

---

## 📊 Dataset
The dataset used in this project includes:
- Financial transaction data for fraud detection
- GitHub pull request data for code modification analysis

---

The models demonstrate effective fraud detection capability, with improved performance achieved through ensemble techniques and neural network optimization.
